# Getting Started

## Installation

Install the package using `uv`:

```console
$ uv add ospsd-team-2
```

Or with pip:

```console
$ pip install ospsd-team-2
```

## Requirements

- Python 3.14+

## Basic Usage

```python
import ospsd_team_2
```
